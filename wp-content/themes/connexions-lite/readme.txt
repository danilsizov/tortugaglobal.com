== Installation ==
1. Go to Appearance >> Add New
2. Search for Fullscreen Lite.
3. Click Install and finally activate the theme.
		Or
1. Download the latest version from https://wordpress.org/themes/fullscreen-lite/
2. Go to Appearance >> Add New >> Upload Theme
3. Select the downloaded theme zip file and click Install Now

== Frequently Asked Questions ==
= How to set Theme Front Page? =
1. Select "A static page" to show on from "Settings -> Reading -> Front page displays"
2. To Write Page Content
	Go to "Appearance -> Cusomize -> Fullscreen Lite Options -> Home Landing Page Section"
	Add landing page section content here and follow below instructions for setting menu.
3. To set the Navigation
	Go to "Appearance -> Menus"
	>>For First Section :- Create a custom link in Menus and put #section1 in URL and Navigation Label according to you for landing page.
	>>For Second Section :- Create a custom link in Menus and put #section2 in URL and Navigation Label according to you for landing page.
	>>For Third Section :- Create a custom link in Menus and put #section3 in URL and Navigation Label according to you for landing page.

= How to create a full width page? =
1. Go to Pages.
2. Click Add New or Edit an existing page.
3. Select "Full Width Template" from "Page Attributes >> Template".

= How to use theme in different languages? =
The Fullscreen Lite is ready for translation. Translation files are in /languages.
Read the following blog for more https://sketchthemes.com/blog/effective-steps-to-translate-your-wordpress-theme-using-qtranslate/


== Upgrade Notice ==
As per the new WordPress guidelines it is required to use the Customizer for implementing theme options. So if you're using an older version then immediately after upgrade go to "Appearance -> Migrate Options" and click on "Migrate to Customizer".
